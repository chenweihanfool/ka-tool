段代碼類型座標系統備註

支援的檔案格式

  • .COA — 界址點座標（點號 + N + E）
  • .BNP — 宗地邊界點序列
  • .PAR — 宗地面積
  • .CTL — 圖根點（格式：點名 NNNNNNN.NNNEEEEEE.EEE）

座標系統

  • 地籍坐標（TWD67）→ EPSG:3828 → 轉換至 EPSG:3826
  • TWD97 → EPSG:3826（不做轉換）
  • 各段代碼對應的座標系統可在「段代碼設定」頁自訂

輸出格式

  • GeoJSON with CRS tag urn:ogc:def:crs:EPSG::3826
  • QGIS 可直接開啟，匯入時選擇 EPSG:3826

瀏覽器需求

  • 需要 Chrome 或 Edge（File System Access API）
  • 需要網路連線（proj4.js CDN）
